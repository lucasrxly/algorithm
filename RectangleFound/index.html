从一组线段中找到一个矩形
本问题是来源与2016年网易有道中校招算法题目

问题描述：现给定一组线段，线段中只包含垂直线段和水平线段，并且线段组中必然存在4条线段能够组成一个完整的矩形，现要求获得这一组线段，并输出。

样例输入：每一行代表一条线段

0 0 0 1        //线段 (0, 0) (0, 1)
0 1 1 1        //线段 (0, 1) (1, 1)
1 1 1 0        //线段 (1, 1) (1, 0)
0 0 1 0        //线段 (0, 0) (1, 0)

思路分析：假设线段为 An = { a1, a2,…, an }，并且是以角标从小到大的方式进行输入的，那么 An 的解必然是 An-1 的解或者 an 必然是矩形中的一条线段，也就是有：

P[An] = P[An-1] 或者 P[An] = Rec[ai, ak, aj, an] 其中 i, k , j < n

写到这里突然想到，其实只要对所有边进行遍历，每次以一条边作为矩形中的一条边，然后通过边的其中一个点进行三次查找，每次找到与该点（集）相连的点集，并将该点集作为下一次查找的点集，得到最后一次查找的点集。
如果最后一次查找的点集中存在一个点与该边的另外一个点重合，则说明该边组成的环即为所求矩形。
